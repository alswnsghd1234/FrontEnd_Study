폼 관련 태그

form 태그 내의 submit버튼 클릭 시 해당 form 내에 작성된
사용자가 입려한 값들을 서버로 넘기면서 요청하는 역할

from action="" method=""
*action 속성 : 해당 form내에 사용자가 입력한 값들을 전달하면서 요청할 서버의 경로들 제시한다
*method 속성 : 요청 전송 방식을 지정하는 속성(get(기본값)/post)
>get 방식: 요청시 사용자가 입력한 값들이 url에 노출되는 방식 , 조회를 할 때 사용
>post 방식 : 요청시 사용자가 입력한 값들이 url에 노출되지 않는 방식, 변경 또는 추가할 때 사용
form태그 내의 제출(submit) 버튼 클릭시 action에 지정한 서버로 method에 작성한
요청 전송 방식으로 사용자가 입력한 값들을 key=value 세트로 전달한다.
