CSS 기본 선택자

선택자란?

특정 html요소를 선택하고자 할 때 사용하는 기능
해당 요소를 선택해서 원하는 스타일과 기능을 적용시킬수 있다.

1.모든(전체) 선택자(*)

모든 요소를 선택하고자 할 때 사용한다. 

      *{
        스타일 속성 : 값;
        스타일 속성 : 값;
      }

2.태그 선택자 (태그명)

해당 문서 내에 같은 태그들을 모두 선택할 때 사용 

      태그명{
        스타일속성 : 값;
      }

3.아이디 선택자 (#아이디명)

해당 문서내에 특정 html 요소 하나만을 선택 할 때 사용 

      #아이디명{
        스타일속성: 값;
      }

4. 클래스 선택자(.클래스명)

해당 문서내에 내가 원하는 여러개의 요소를 선택하고자 할 때 사용
클래스 속성을 추가해준다. 

      .클래스명{
        스타일속성: 속성값;
      }
