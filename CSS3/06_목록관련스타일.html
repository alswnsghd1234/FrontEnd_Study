목록 관련 스타일

list-style-type : 불릿기호를 변경시켜줄 때 사용

    *순서 없는 목록(ul)
    선택자{
      list-style-type:disc/circle/square/none;

    }

    *순서 있는 목록(ol)
      list-style-type:decimal/decimal-leading-zere;
      list-style-type:lower-alpha/upper-alpha;
      list-style-type:lower-alpha/upper-roman;

순서 없는 목록

  • HTML
  • CSS
  • JavaScript
  • jQuery

순서 있는 목록

  1. HTML
  2. CSS
  3. JavaScript
  4. jQuery

list-style-position:불릿기호의 위치 조정

list-style-image:불릿기호를 이미지 파일로

  선택자{
    list-style-position: inside/outside;
    list-style-image:url(적용시키고자하는 이미지의 경로)
  }
