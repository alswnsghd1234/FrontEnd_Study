배열

자바스트립트에서는 변수 선언 시 별도의자료형 지정이 없기 때문에
어떤 자료형의 값이 오던 한 배열에 담을 수 있다(자바의 컬렉션과 유사)



배열의 선언

배열선언 시 배열의 크기를 지정한 채로 선언하거나, 크기를 지정하지 않아도 된다.
배열의 크기 제약이 없음

Array객체의 메소드

1) index(찾고자하는요소):배열에서 해당요소가 위치해있는 인덱스 반환

2) concat(배열1,배열2,...) : 여러개의 배열을 결합하고자 할 때 사용

3) reverse() : 배열에 담긴 값들을 역순으로 바꿔주는 메소드

4) sort() : 배열 안에 달린 값들을 오름차순으로 정렬시켜주는 메소드

5_1)push(추가할 요소) : 배열 맨 뒤에 요소를 추가하고 배열의 크기 반응 5_2)pop() : 배열의 맨 뒤 요소를 제거하고 제거된 요소를 반환
